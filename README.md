# eBook
This is a eBook website based on [Wagtail](https://wagtail.org/).
## Getting started
Python 3.8
```bash
git clone https://github.com/Jiayi-Zeng/eBook.git
cd eBook
pip install -r requirements.txt
python manage.py runserver
```
